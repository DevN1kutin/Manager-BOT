# 🦾 Телеграм бот (Менеджер)

Полноценный бот для управления группами с функциями модерации и развлечений.

---

##  Начало

### 1. Установи зависимости
```bash
pip install -r requirements.txt
```

### 2. Получи токен
- Открой [@BotFather](https://t.me/BotFather) в Telegram
- Напиши `/newbot` и следуй инструкциям
- Скопируй полученный токен

### 3. Вставь токен в `config.py`
```python
BOT_TOKEN = "1234567890:ABCdef..."
```

### 4. Запусти бота
```bash
python bot.py
```

### 5. Добавь бота в группу
- Добавь бота в свою группу
- Выдай ему права **администратора** (нужно для бана, мута, кика)

---

## 👮 Команды администратора

| Команда | Описание |
|---------|----------|
| `/ban @user` | Забанить пользователя |
| `/unban @user` | Разбанить пользователя |
| `/mute @user 10m причина` | Замутить на время (s/m/h/d) |
| `/unmute @user` | Размутить |
| `/kick @user` | Кикнуть из чата |
| `/warn @user причина` | Предупреждение (3 = бан) |
| `/pin` | Закрепить (ответом на сообщение) |
| `/unpin` | Открепить |
| `/ro` | Режим только для чтения |
| `/ro off` | Выключить режим только чтения |

## ℹ️ Информация

| Команда | Описание |
|---------|----------|
| `/info` | Инфо о пользователе |
| `/chatinfo` | Инфо о чате |

## 💬 Диалог

Бот умеет отвечать на обычные сообщения:
- В **личке** — всегда
- В **группе** — если упомянуть `@бота` или написать слова-триггеры

Триггеры: приветствия, "как дела", "спасибо", "скучно" и другие.

Убедись, что у бота есть следующие права:

- ✅ Блокировка пользователей
- ✅ Ограничение участников

- ✅ Управление чатом
